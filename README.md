## 🖥️ О программе

**Риэлтор Профи** — это desktop-приложение для риэлторов,
которое автоматически собирает объявления об аренде квартир
из групп ВКонтакте и помогает управлять ими.

Больше не нужно вручную просматривать десятки групп —
программа сделает это за вас и красиво отобразит
все объявления с фото, ценами и контактами.

```
┌─────────────────────────────────────────────┐
│  ВКонтакте группы  →  Программа  →  Работа  │
│                                             │
│  • Аренда СПб         • Фото галерея        │
│  • Сдам квартиру  →   • Телефоны       →   │
│  • Жильё Москва       • Адреса              │
│                        • Заметки            │
└─────────────────────────────────────────────┘
```

## ⚙️ Установка

### Вариант А — Запуск EXE (для всех)

```
1. Скачайте и распакуйте архив RealtorPro.zip
2. Запустите RealtorPro.exe
3. Готово — переходите к настройке токенов
```

### Вариант Б — Запуск через Python (для разработчиков)

**Шаг 1.** Убедитесь что установлен Python 3.9+

```bash
python --version
# Должно показать: Python 3.9.x или выше
```

Если Python не установлен → [скачать с python.org](https://python.org/downloads)

> ⚠️ При установке Python обязательно поставьте галочку
> **"Add Python to PATH"**

---

**Шаг 2.** Скачайте программу

```bash
git clone https://github.com/Berhamon/Realtor.git
cd realtor-app
```

Или скачайте ZIP на GitHub → кнопка **Code → Download ZIP**

---

**Шаг 3.** Создайте виртуальное окружение

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Если появилась ошибка с activate — выполните сначала:
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

```bash
# Linux / macOS
python3 -m venv venv
source venv/bin/activate
```

---

**Шаг 4.** Установите зависимости

```bash
pip install -r requirements.txt
```

Установка займёт 1-3 минуты. Вы увидите скачивание пакетов.

---

**Шаг 5.** Запустите программу

```bash
python main.py
```

## 🔑 Получение VK токена

VK токен нужен чтобы программа могла читать посты из групп ВКонтакте.

> 💡 Токен привязан к вашему аккаунту ВКонтакте.
> Никому не передавайте его — это как пароль от аккаунта.

---

### 📌 Способ 1 — Самый простой (рекомендуется)

**Шаг 1.** Войдите в ВКонтакте в браузере

**Шаг 2.** Скопируйте одну из ссылок и откройте в браузере:

```
https://oauth.vk.com/authorize?client_id=2685278&display=page&redirect_uri=https://oauth.vk.com/blank.html&scope=groups,wall,photos,offline&response_type=token&v=5.131
```

> Это официальное приложение Kate Mobile — безопасно

**Шаг 3.** Нажмите кнопку **"Разрешить"**

```
┌──────────────────────────────────┐
│  Kate Mobile запрашивает доступ  │
│                                  │
│  • Доступ к стене                │
│  • Доступ к группам              │
│  • Доступ к фото                 │
│                                  │
│  [ Разрешить ]  [ Отмена ]       │
└──────────────────────────────────┘
```

**Шаг 4.** Страница станет белой — это нормально!

Посмотрите на **адресную строку браузера**:

```
https://oauth.vk.com/blank.html#access_token=vk1.a.ЗДЕСЬ_ВАШ_ТОКЕН&expires_in=0&...
                                              ↑─────────────────────↑
                                              Копируйте только это!
```

**Шаг 5.** Скопируйте всё что идёт после `access_token=` и до `&expires_in`

```
Пример токена:
vk1.a.AbCdEfGhIjKlMnOpQrStUvWxYz1234567890abcdefghijklmnopqrstuvwxyz
```

**Шаг 6.** Вставьте токен в программе:
```
⚙️ Настройки → поле "VK Access Token" → Сохранить
```

---

### 📌 Способ 2 — Альтернативные ссылки

Если Способ 1 выдаёт ошибку — попробуйте другие ссылки:

**VK для iPhone:**
```
https://oauth.vk.com/authorize?client_id=3140623&display=page&redirect_uri=https://oauth.vk.com/blank.html&scope=groups,wall,photos,offline&response_type=token&v=5.131
```

**VK для Android:**
```
https://oauth.vk.com/authorize?client_id=2274003&display=page&redirect_uri=https://oauth.vk.com/blank.html&scope=groups,wall,photos,offline&response_type=token&v=5.131
```

---

### 📌 Способ 3 — Создать своё приложение

Используйте если все предыдущие способы не работают.

**Шаг 1.** Перейдите на [dev.vk.com](https://dev.vk.com)

**Шаг 2.** Нажмите **"Подключить приложение"**

```
┌─────────────────────────────────┐
│      Создание приложения        │
│                                 │
│  Название: MyRealtorParser      │
│  Тип: ● Мини-приложение         │
│                                 │
│         [ Создать ]             │
└─────────────────────────────────┘
```

**Шаг 3.** После создания перейдите в **Настройки** приложения

**Шаг 4.** Скопируйте **ID приложения** (число, например: `12345678`)

**Шаг 5.** Вставьте ваш ID в ссылку и откройте её:

```
https://oauth.vk.com/authorize?client_id=ВАШ_ID&display=page&redirect_uri=https://oauth.vk.com/blank.html&scope=groups,wall,photos,offline&response_type=token&v=5.131
```

**Шаг 6.** Разрешите доступ → скопируйте токен из адресной строки

---

### ✅ Проверка токена

Вставьте в браузер (замените TOKEN на ваш):
```
https://api.vk.com/method/users.get?access_token=TOKEN&v=5.131
```

Правильный ответ выглядит так:
```json
{
  "response": [{
    "id": 123456789,
    "first_name": "Иван",
    "last_name": "Иванов"
  }]
}
```

Если видите своё имя — токен работает ✅

Если видите ошибку — попробуйте другой способ получения ❌

## 🐙 Настройка GitHub

GitHub используется для хранения ваших данных (квартиры, заметки).
Благодаря этому вы можете открыть программу на другом компьютере
и все данные будут на месте.

> 💡 GitHub — бесплатный сервис. Регистрация займёт 2 минуты.
> Ваши данные хранятся в приватном (закрытом) репозитории.

---

### Шаг 1 — Регистрация на GitHub

Если у вас нет аккаунта:

```
1. Перейдите на https://github.com
2. Нажмите "Sign up" (Зарегистрироваться)
3. Введите email, пароль, имя пользователя
4. Подтвердите email
```

---

### Шаг 2 — Создание токена GitHub

**1.** Войдите на [github.com](https://github.com)

**2.** Нажмите на ваш **аватар** (фото профиля) в правом верхнем углу

**3.** В выпадающем меню выберите **"Settings"** (Настройки)

```
┌─────────────────┐
│  ВашЛогин       │
│  ─────────────  │
│  Your profile   │
│  Settings    ←  │  ← Нажать сюда
│  ...            │
└─────────────────┘
```

**4.** Прокрутите левое меню вниз до конца → нажмите **"Developer settings"**

```
Левое меню GitHub Settings:
┌─────────────────────────┐
│  Account                │
│  Appearance             │
│  ...                    │
│  ─────────────────────  │
│  Developer settings  ←  │  ← Нажать сюда (самый низ)
└─────────────────────────┘
```

**5.** Нажмите **"Personal access tokens"** → **"Tokens (classic)"**

```
Developer Settings:
┌──────────────────────────────────┐
│  GitHub Apps                     │
│  OAuth Apps                      │
│  Personal access tokens       ↓  │
│  ├── Fine-grained tokens         │
│  └── Tokens (classic)        ←   │  ← Нажать сюда
└──────────────────────────────────┘
```

**6.** Нажмите **"Generate new token"** → **"Generate new token (classic)"**

**7.** Заполните форму:

```
┌─────────────────────────────────────────┐
│  New personal access token (classic)    │
│                                         │
│  Note (название):                       │
│  ┌──────────────────────────────────┐   │
│  │ realtor-pro-app                  │   │
│  └──────────────────────────────────┘   │
│                                         │
│  Expiration (срок действия):            │
│  ┌──────────────┐                       │
│  │ No expiration│  ← Выбрать это        │
│  └──────────────┘                       │
│                                         │
│  Select scopes (права доступа):         │
│  ☑ repo          ← ОБЯЗАТЕЛЬНО          │
│    ├─ repo:status                       │
│    ├─ repo_deployment                   │
│    ├─ public_repo                       │
│    └─ repo:invite                       │
│                                         │
│  [ Generate token ]                     │
└─────────────────────────────────────────┘
```

> ⚠️ Обязательно поставьте галочку **"repo"** — 
> без неё программа не сможет сохранять данные!

**8.** Нажмите зелёную кнопку **"Generate token"**

**9.** ❗ ВАЖНО: Токен показывается **ТОЛЬКО ОДИН РАЗ!**

```
┌─────────────────────────────────────────────┐
│  ✅ Ваш новый токен:                         │
│                                             │
│  ghp_AbCdEfGhIjKlMnOpQrStUvWxYz1234        │
│  ─────────────────────────────────          │
│  [ 📋 Скопировать ]                         │
│                                             │
│  ⚠️ Сохраните токен сейчас!                 │
│     После закрытия страницы                 │
│     он больше не будет виден.               │
└─────────────────────────────────────────────┘
```

Скопируйте токен и сохраните в надёжное место (например, в заметки).

### 🔄 Синхронизация между устройствами

Ваши данные автоматически сохраняются на GitHub после каждого действия.

**Чтобы открыть программу на другом компьютере:**

```
1. Установите программу (EXE или Python)
2. Откройте ⚙️ Настройки
3. Введите те же токены VK и GitHub
4. Нажмите "🔄 Синхронизировать"
5. Все квартиры и заметки загрузятся автоматически
```

**Кнопка "🔄 Синхронизировать"** в шапке программы:
- Загружает данные с GitHub
- Объединяет с локальными изменениями
- Сохраняет обратно на GitHub

## ❓ Частые вопросы

<details>
<summary><b>❓ Почему программа не находит объявления?</b></summary>

Возможные причины:
1. **Неверный VK токен** — проверьте в настройках, попробуйте получить новый
2. **Нет групп** — добавьте хотя бы одну группу в настройках
3. **Группа закрытая** — программа работает только с публичными группами
4. **В группе нет объявлений об аренде** — программа фильтрует только посты про аренду
</details>

<details>
<summary><b>❓ Почему адрес/цена/телефон не определяется?</b></summary>

Программа извлекает данные автоматически из текста.
Если автор написал объявление нестандартно — данные могут не определиться.

В таком случае:
- Откройте оригинальный пост кнопкой **"🔗 ВКонтакте"**
- Найдите данные вручную
- Для одобренных квартир запишите в заметки
</details>

<details>
<summary><b>❓ GitHub не подключается (показывает 🔴)?</b></summary>

Проверьте по порядку:
1. Правильно ли скопирован токен (без пробелов)
2. Правильно ли написан логин GitHub (регистр важен)
3. Правильно ли написано название репозитория
4. Есть ли интернет-соединение
5. Нажмите "Сохранить" и затем "Проверить соединение"
</details>

<details>
<summary><b>❓ Данные сохраняются если нет интернета?</b></summary>

Да! При отсутствии интернета данные сохраняются локально
в файл `local_data.json` рядом с программой.
При следующем подключении синхронизируйте вручную кнопкой
**"🔄 Синхронизировать"**.
</details>

<details>
<summary><b>❓ Фотографии не загружаются?</b></summary>

1. Проверьте интернет-соединение
2. Подождите 5-10 секунд (загрузка асинхронная)
3. Фото кэшируются — при повторном открытии загрузятся быстрее
4. Если фото нет совсем — в объявлении не было фотографий
</details>

<details>
<summary><b>❓ Как запустить программу на другом компьютере?</b></summary>

**EXE версия:**
```
1. Скопируйте всю папку RealtorPro на другой компьютер
2. Запустите RealtorPro.exe
3. Введите токены в настройках
4. Нажмите "Синхронизировать"
```

**Python версия:**
```
1. Установите Python и зависимости
2. Скопируйте папку проекта
3. Запустите python main.py
4. Введите токены в настройках
```
</details>

<details>
<summary><b>❓ Антивирус блокирует EXE файл?</b></summary>

Это ложное срабатывание — EXE создан через PyInstaller
(стандартный инструмент для Python приложений).
Программа не содержит вирусов.

Решение:
1. Добавьте файл в исключения антивируса
2. Или используйте Python версию (запуск через `python main.py`)
</details>

## 🛠️ Решение проблем

### Программа не запускается

```
Симптом: Двойной клик на EXE — ничего не происходит

Решение:
1. Убедитесь что распаковали весь архив, не только EXE
2. Попробуйте запустить от имени администратора
   (правая кнопка → "Запуск от имени администратора")
3. Проверьте что антивирус не заблокировал файл
```

### Ошибка при парсинге

```
Симптом: Статус показывает "❌ Ошибка: ..."

Решение:
1. Проверьте VK токен — возможно истёк
2. Получите новый токен (Способ 1 из инструкции)
3. Убедитесь что ссылки на группы корректны
4. Проверьте интернет-соединение
```

### Данные не синхронизируются

```
Симптом: В шапке показывает 🔴 GitHub: ошибка

Решение:
1. Перейдите в ⚙️ Настройки
2. Проверьте все три поля GitHub (токен, логин, репозиторий)
3. Нажмите "Сохранить все настройки"
4. Нажмите "Проверить соединение"
5. Если ошибка — создайте новый GitHub токен
```

### Программа тормозит

```
Симптом: Лаги при вводе в поиск, зависание

Решение:
1. Уменьшите количество групп для парсинга
2. Используйте фильтры — не держите 500+ карточек открытыми
3. Перезапустите программу
```
